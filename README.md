# Dashboard de Sostenibilidad – Empresas Gasco

## Descripción
Dashboard interactivo para visualizar indicadores ESG y métricas de sostenibilidad.

## Tecnologías
- HTML5
- TailwindCSS
- JavaScript
- Chart.js

## Cómo usar
1. Abrir el archivo index.html en el navegador.
2. Hacer clic en "Comenzar experiencia".
3. Navegar por las secciones del dashboard.

## Estructura de carpetas
- /IMG-VID → Logos e imágenes
- index.html → Archivo principal
